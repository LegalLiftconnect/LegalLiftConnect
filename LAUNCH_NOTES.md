## What This App Includes

### Public User Side
- Check My Case
- Enter Accident Information
- Upload Documents & Photos
- Consent checkbox
- Case tracking ID after submission

### Partners & Referrals
- Become a Partner form
- Submit a Referral form
- Partner categories:
  - Tow Truck Company
  - Body Shop
  - Chiropractor
  - Physical Therapy Clinic
  - Medical Provider
  - Community Partner
  - Other

### Admin Dashboard
- Admin login
- View all leads
- View referrals
- View partners
- Add law firms
- Assign leads to law firms
- Change lead status:
  - New Lead
  - Contacted
  - Verified
  - Sent to Firm
  - Accepted
  - Signed Case

### Law Firm Portal
- Law firm login
- View assigned leads only
- View accident details
- View uploaded documents
- Accept/reject leads
- Request more information
- Mark signed case
- Add firm notes

## How To Run Locally

1. Install Node.js
2. Open this folder in Terminal
3. Run:

npm install

4. Start the app:

npm start

5. Open browser:

http://localhost:3000

## Before Public Launch

Do these before putting real customer information into the app:

- Replace demo passwords with secure user accounts
- Add HTTPS/SSL
- Use secure cloud hosting
- Add privacy policy
- Add terms and conditions
- Add stronger consent language
- Add database backups
- Add email notifications to info@legalliftgroup.com
- Add SMS notifications
- Consult an attorney about referral/lead rules in Georgia and Florida
- Consult a privacy/security professional if collecting medical documents
